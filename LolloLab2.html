Lab 2: Functions of the Catalog in Zotero

Find a known item:

  • Item Type
  • Title and/or Short Title (only in the Ardizzone and Dorn items)
  • Author

Find a set of items:

  • Item Type
  • Publisher
  • Catalog

Identify:

  • Item Type
  • Edition (only in the Ardizzone item)
  • ISBN
  • Call Number
  • Publication Date
  • Publisher
  • Language (only in the Dorn item)

Select:

  • ISBN
  • Call Number
  • Edition (integral for users in academic libraries especially)

Obtain:

  • ISBN
  • Call Number

I’d propose an extra field for “Location” here, since it would be integral for library users to obtain an item in its physical location if they don’t order it online from the library catalog.

Navigate: I specifically took this to mean the search function on the library site as well as the searching of specific library locations

  • Item Type (because they are all indexed in this way)
  • Title
  • Author

I’d also propose the “Location” field here, as having the specific library location indexed could support the user’s experience.
